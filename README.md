# Cephalon Cy - многофункциональный чат-бот

Развлекательно-утилитарный чат-бот для популярного мессенджера, написанный на Python с использованием [aiogram](https://docs.aiogram.dev/).  
Предоставляет интерактивную игру, генератор случайных чисел, информационные команды и встроенную систему помощи.

## Возможности

### Игры
- **Connect4 («Четыре в ряд»)** - игра против бота с управлением через инлайн-кнопки. Поле 6×7, подсвеченное эмодзи, проверка победы по всем направлениям.

### Генераторы
- `/roll` - генератор случайных чисел:
  - без аргументов - число от 0 до 100
  - число `N` - от 0 до N
  - диапазон `X-Y` - от X до Y
  - формат `KdM` - бросок K кубиков с M гранями (до 20 кубиков/граней), с суммой под спойлером
- `/coin` - подбрасывание монетки (Орёл / Решка)

### Полезные команды
- `/about` (и `/me`) - информация о пользователе: имя, ID, username, язык, аватар
- `/avatar` - аватар пользователя
- `/chatinfo` (или `/chat`, `/serverinfo`, `/guild`) - сведения о чате: тип, название, ID, разрешения

### Навигация
- `/help` - многоуровневая справка с кнопками по разделам и подсказками по каждой команде.

## Стек

- **Язык**: Python 3.12
- **Фреймворк**: aiogram 3.10 (асинхронный)
- **Хранение**: без внешней БД - состояние игр хранится в оперативной памяти (словарь `states`), для FSM предусмотрен `MemoryStorage`
- **Конфигурация**: токен и чувствительные данные вынесены в `.env` (через python-dotenv)
- **Архитектура**: модульная - обработчики команд разделены по файлам в пакете `handlers/`, динамическая загрузка

## Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/D-9341/tg-Bot.git
   cd tg-Bot
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл vars.env в корне проекта с содержимым:
   ```bash
   TOKEN=ТОКЕН_БОТА
   ```

4. Запустите бота:
   ```bash
   python main.py
   ```
Бот запуститься в режиме long polling
